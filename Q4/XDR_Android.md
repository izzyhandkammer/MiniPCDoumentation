# XDR on Android
### 1. Configure and Start Emulator
- Download the Android Studio app to spin up your mobile environment
- Create a new project and select "Empty Activity"
- Choose API 34 and start the project
- On the right hand device manager:
	- Add device -> new virtual device
	- I chose Pixel 7
	- Choose the latest API and make sure google play store is selected by default
	- Launch it with the green play button

### 2. Download the XDR App
- On the XDR console, right clicking on the installation will give you the option to "View Installation Links". Copy it and paste it in the browser of the emulator.
- Verify install on the XDR Inventory.

### 3. Testing
- For testing, I chose to navigate to the [EICAR](https://www.eicar.org/download-anti-malware-testfile/) website and download a test file.
- Attempting to open the file should be blocked by the app.
- Verify in the XDR Case dashboard.
